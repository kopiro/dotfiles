## kopiro dotfiles

```
cd /opt && git clone git@github.com:kopiro/dotfiles.git
```

```
./init.sh
```

```
./conf.sh
```

```
./macos.sh

```
./zsh.sh
```

### titanium

```
./titanium.sh
